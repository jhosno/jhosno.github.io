challenges:
  - challenge: Integración API Externa + DB Local - Combinar datos de TheDogApi con razas creadas localmente manteniendo una experiencia unificada
    solution: Implementación de capa de normalización de datos en el backend que unifica estructura de respuestas de ambas fuentes, con flag de origen para diferenciar fuente cuando sea necesario.
  - challenge: Gestión de Estado Complejo - Sincronizar múltiples filtros, búsqueda y ordenamiento sin perder performance ni causar renders innecesarios
    solution: Arquitectura Redux con slices específicos para filtros, reducers que componen lógica de filtrado/ordenamiento y selectors memoizados para computación eficiente.
  - challenge: Diseño Visual sin Frameworks CSS - Crear interfaz atractiva y responsive usando únicamente CSS vanilla sin dependencias externas
    solution: Sistema de variables CSS custom para theming consistente, arquitectura BEM para nomenclatura escalable y uso estratégico de Grid/Flexbox para layouts responsivos.
  - challenge: Validación de Formularios - Asegurar integridad de datos al crear nuevas razas con feedback claro al usuario
    solution: Validaciones en frontend con mensajes contextuales en tiempo real y validaciones adicionales en backend antes de persistir en DB.
